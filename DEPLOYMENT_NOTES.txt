AIRS Website Deployment Notes

1. Install and test locally:
   npm install
   npm run dev

2. Build production files:
   npm run build

3. Preview production build locally:
   npm run preview

4. GoDaddy deployment:
   - Open GoDaddy/cPanel File Manager.
   - Back up current public_html first.
   - Upload the contents of the dist folder into public_html.
   - index.html must be directly inside public_html.
   - assets/ must be directly inside public_html.

5. Formspree:
